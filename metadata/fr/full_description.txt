EtchDroid est une application open source qui vous aide à écrire des images sur des clés USB.
Utilisez-le pour créer une clé USB amorçable lorsque votre ordinateur portable ne marche plus.

Appareils pris en charge: 
✅ Clé USB
✅ Adaptateur carte SD USB
❌ Disques durs USB/SSD 
❌ Hubs USB
❌ Carte micro SD interne
❌ CD/DVD/Disquettes
❌ Appareils thunderbolt

Prévu :
• Écrire une distribution GNU / Linux commune en streaming depuis Internet sans la stocker sur l'appareil
• Prise en charge de l'installation ISO de Windows (nécessitera probablement la racine)
• Prise en charge du clignotement d'image de type Unetbootin

Le code source est sur GitHub : https://github.com/Depau/EtchDroid
